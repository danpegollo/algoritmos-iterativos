# Estruturas de Repetição e Sistemas com While em Python

Repositório com exercícios desenvolvidos para praticar estruturas de repetição utilizando o laço `while` em Python.

## Exercícios

### 1. Série Matemática com Potências
Programa que realiza operações matemáticas utilizando potência, divisão e alternância entre soma e subtração.

### 2. Soma de Sequência Numérica
Programa que calcula uma sequência de divisões e acumula os resultados progressivamente.

### 3. Sistema de Funcionários
Programa que realiza cadastro de funcionários, valida dados, calcula reajustes salariais e exibe estatísticas da empresa.

### 4. Sistema de Votação
Programa que simula uma votação, contabiliza votos dos candidatos, votos nulos e votos em branco, além de calcular percentuais.

## Assuntos praticados

- Laço `while`
- Estruturas de repetição
- Loops aninhados
- Condicionais (`if`, `elif`, `else`)
- Contadores
- Acumuladores
- Operadores matemáticos
- Porcentagem
- Média aritmética
- Validação de dados
- Entrada e saída de dados
- Lógica de programação
