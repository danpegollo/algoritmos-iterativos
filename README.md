# Exercícios de Python — Estruturas de Repetição

Repositório com exercícios desenvolvidos em Python para praticar lógica de programação e estruturas de repetição.

## Conteúdo

Os exercícios abordam:
- Estruturas `while`
- Variáveis acumuladoras
- Contadores
- Operações matemáticas
- Validação de dados
- Condições (`if`, `elif`, `else`)
- Porcentagem
- Média aritmética
- Manipulação de entradas do usuário

---

# Exercícios

## Exercício 1
Programa que:
- Recebe um número inteiro
- Realiza operações matemáticas com potências e divisões
- Utiliza uma sequência alternando soma e subtração

Conceitos praticados:
- Potenciação
- Estrutura `while`
- Operadores matemáticos

---

## Exercício 2
Programa que:
- Calcula uma sequência de divisões
- Soma os resultados progressivamente
- Exibe o resultado final formatado

Conceitos praticados:
- Acumuladores
- Divisão
- Incremento de variáveis

---

## Exercício 3 — Sistema de Funcionários
Programa que:
- Recebe informações de funcionários
- Valida sexo e salário
- Calcula reajustes salariais
- Conta funcionários masculinos e femininos
- Calcula média salarial
- Calcula percentual de mulheres

Conceitos praticados:
- Validação de dados
- Média
- Porcentagem
- Estruturas condicionais

---

## Exercício 4 — Sistema de Votação
Programa que:
- Simula uma votação
- Conta votos de candidatos
- Conta votos nulos e brancos
- Calcula percentuais

Conceitos praticados:
- Contadores
- Estrutura de repetição
- Percentuais
- Condicionais
